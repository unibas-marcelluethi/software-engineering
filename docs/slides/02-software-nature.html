Was macht ein Ingenieur?

Ziel des Ingenieurs ist ein Produkt zu bauen.
  • Das Produkt Software ist irgendwie anders!

Eigenschaften von Software (1)

Software ist formbar (engl. malleable)
  • Das Produkt lässt sich einfach verändern
  • Das Design ist jedoch starr

Gefahr von Veränderungen

  • Software wird verändert ohne das Design zu ändern.
  • Produkt und Design divergieren

Eigenschaften von Software (2)

Software ist immateriell. Produktkosten entstehen hauptsächlich durch Entwicklung/Wartung

Eigenschaften von Software (2)

Traditionelle Ingenieursdisziplin

  • Herstellung (manufacturing) grösster Kostenfaktor.
  • Fehler sind teuer
    • Fokus: Reibungslose Produktion garantieren

Softwareengwicklung

  • Herstellung ist trivial (kopieren)
  • Fokus: Design und Implementation
    • Prozess braucht definierte Qualitätskriterien

Eigenschaften von Software (3)

Software folgt nicht den Naturgesetzen
  • Naturgesetze definieren Lösungsraum
    • Geben uns Hinweise was wir nicht versuchen müssen

Software Engineering vs. Physics

"God does not play dice with the universe" - Einstein
  • Naturgesetzte / Physik lassen uns Schlussfolgerungen ziehen
  • Software hat wenig verlässiliche Strukturen
    • Logische Folgerungen über sind schwierig
    • Mathematische Strukturen müssen explizit Designed und Dokumentiert werden

Eigenschaften von Software (4)

Anforderung an Softwareprodukt ist schwierig von Designkriterien zu trennen.
  • Typische Anforderungen:
    • System soll wartbar und erweiterbar sein
    • System soll skalieren
  • Schwierig zu beurteilen ohne das Design zu betrachten
    • Keine Physik, die uns hilft Schlussfolgerungen zu ziehen
Wir brauchen Disziplin und gute Regeln um gute Software zu bauen
  • Was ist denn eigentlich gute Software?

Unit tests

  • Wo muss ich ansetzen um Softwarequalität zu beeinflussen
  • Was sind die Vorteile / Nachteile das Software formbar (malleable) ist?
  • Weshalb ist es problematisch, dass keine mathematischen Eigenschaften (Naturgesetze) existieren, welche die Software beschreiben?
