# Projektübersicht

Ein Teil dieser Vorlesung ist ein Projekt, welches Sie in Gruppen von 2-5 Personen realisieren. 
Die Aufgabe ist es, eine Erweiterung für die Open Source Software [JabRef](https://www.jabref.org/).
